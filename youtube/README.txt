This section will be the youtube interface for the bot.

The YT interface will just be for information purposes, no commands will be processed. Info
notices will be chatted to the livestream chat in YT during the stream.

Info categories:
Announce game started
Announce game paused
Announce game paused
Announce new player cutoff
Announce player added
Announce slot calls (include number of players that matched the call)
Announce bingos
Announce simplified game stats when ended
Announce when a request is made
Periodically announce outstanding requests

Prefix all bot announcements with [BINGO]
